# Sandbox V2 Configuration Examples

These examples show how common Culina framework configuration records relate to one another.

Use them as reference shapes when creating or reviewing jobs:

1. `customer_orders_ingestion_job.json`
2. `customer_orders_transformation_job.json`
3. `customer_orders_dependencies.json`
4. `customer_orders_validation.json`

The examples use placeholder names and sandbox paths. Replace them with environment-specific values when creating real configuration.
